## Vitality Hub
Healthy Lifestyle recommendations

### Features
- Body Mass Index Calculator
- Waist Hip Radio Calculator 
- Personal log with recipes and exercises
- Consumed Calories Tracker 

### Installation
- Clone repo
- Run npm i
- /server rails server
- /client Npm run dev
